# bookworm.py
*A simple Python-based library system.*

## Setup

- Download Python 3+
- Clone this repo
- Run BarcodeGeneration
- Run bookworm.py on a thin client with no keyboard and a barcode scanner, with barcodes titled "Check Out" corresponding to a 13 digit code "9999999999994", titled "Return" and corresponding to "9999999999993" and titled "Exit" corresponding to "9999999999992" glued next to the machine.
